Welcome, Datarun!

  • It is serving REST APIs, under the '/api' URLs.
  • To manage this microservice, you will probably want to use Consul:
    • To run Consul locally, you can use Docker:
      docker compose -f src/main/docker/consul.yml up --wait
    • Its default URL is http://localhost:8500/
  • OpenAPI documentation endpoint for those APIs is at /v3/api-docs, but if you want access to the full Swagger UI, you should use a gateway or a Registry, which will serve as API developer portals.
